# SPIKES Protocol Implementation

## Overview

APEX uses the SPIKES protocol to structure difficult medical conversations.

The system tracks learner progression through the following stages.

## SPIKES Stages

Setting  
Perception  
Invitation  
Knowledge  
Empathy  
Strategy and Summary

## Stage Detection

The system infers stages using rule-based detection of dialogue cues.

Examples:

Setting → greetings and conversation setup  
Perception → asking what the patient understands  
Invitation → asking permission to explain  
Knowledge → diagnostic disclosure  
Empathy → emotional validation  
Strategy → discussion of next steps

## Role in APEX

The SPIKES stage is used for:

• guiding patient responses  
• evaluating learner performance  
• structuring feedback reports
